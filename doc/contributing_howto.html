Keeping up-to-date

The latest code can always be checked out through SVN. See the User's Guide for instructions on how to checkout an up to date version.

IMPORTANT: Copyright and license text

Prior to checking in a new file make sure that the copyright information is accurate, reflects the truth, and that the license text is present.

Here is the text that should appear in every file checked in the PLearn CVS repository on Source Forge:


Copyright (C) YEAR(s) AUTHOR(s)
