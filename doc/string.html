The standard C++ string class and PLearn

We encourage the use of the standard C++ string class within PLearn.

If you are not familiar with the string class, take a look at this quick tutorial.

In addition PLearn defines the following useful functions (in stringutils.h which is included by general.h):

  • tostring which will transform about anything (number, Vec, ...) to a string
  • left, right and center that allow you to align a string within a cell of given size.

These allow you to do things such as: 

string firstname;
int age;
cout << "Please enter your first name and age";
cin >> firstname >> age;
string s = string("Your name is ") + firstname + " and you are " + tostring(age) + " years old.";
string tablerow = center(firstname,20) + "|" + right(tostring(age),3);
